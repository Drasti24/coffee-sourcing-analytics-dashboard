# ☕ Coffee Sourcing & Roasting Decision Intelligence Dashboard

## 📌 Project Overview

This project analyzes coffee sourcing profitability, roast-level performance, demand trends, and revenue generation across multiple global coffee origins using SQL Server and Power BI.

The dashboard was inspired by conversations with coffee producers and roasters at a coffee collective event, where I learned how complex and data-driven sourcing and roasting decisions are within the specialty coffee industry.

The goal of this project was to build a business-focused analytics dashboard that supports sourcing and roasting strategy decisions through data visualization and profitability analysis.

---

## 🛠 Tools & Technologies Used

- SQL Server Management Studio (SSMS)
- SQL
- Power BI
- Data Visualization
- Business Intelligence & Dashboard Design

---

## 📊 Key Business Questions Answered

- Which coffee origins generate the highest revenue?
- Which origins are the most profitable?
- Which roast level performs best in terms of demand and profit?
- Which origin-roast combinations produce the strongest profitability?
- Which coffee origins maintain the highest profit margins?

---

## 📈 Dashboard Features

- Executive KPI Overview
- Profitability Heatmap by Origin & Roast Combination
- Revenue Analysis by Coffee Origin
- Profit Margin Analysis
- Roast-Level Demand & Profit Analysis
- Coffee Origin Profit Distribution
- Business Objective & Strategic Insights

---

## 🧠 Key Insights

- Brazil generated the highest overall revenue and profitability
- Medium roast consistently outperformed Light and Dark roasts in both demand and profit
- Ethiopia demonstrated strong premium-origin positioning
- Honduras maintained strong profit margins despite lower total demand
- Profitability varied significantly across origin and roast combinations
